## Weather APP
This app shows the weather in the near future for Graz.
Api used:(https://dataset.api.hub.geosphere.at/v1/docs/index.html)

# Goals:
 - Get Data
 - Style with Tailwind
 - Make it customizable which data you see
 - Maybe add another API

```bash
# install dependencies
npm i

# run app
npm run dev
```
